## objects
```
objects används för att spara data
```
```
let name = "Jonas";
let age = 30;
```
```
// creating an empty object
let user = new Object();
let user = {};

let user = {
    name: "Jonas"
    age: 30,
};
```
```
// Accessing data in a object
alert(user.name); // value = Jonas
alert(user.age); // value = 30
```
